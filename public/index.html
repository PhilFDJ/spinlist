Spinlist

Crowd-sourced setlists

Let the room
pick the music.

Hosts build an event, share one link or QR code, and guests search Spotify's real catalogue to vote for the songs they want. When voting closes, the DJ gets a ranked playlist — PDF and Spotify export.

01 / HOST

Build the event

Name it, set a vote deadline, choose how many votes each guest gets.

02 / SHARE

One link, one code

Send a link or print the QR for tables. No app, no login for guests.

03 / VOTE

Guests choose

Search Spotify live, vote for favourites, watch the leaderboard move.

04 / EXPORT

Hand to the DJ

At the deadline, download a ranked PDF and push a Spotify playlist.
