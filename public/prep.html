Match the couple's song requests against your own music library, then export a playlist (.m3u) for each block to drop straight into your DJ software. Your files stay on your computer — nothing is uploaded.

📁 Scan a folder
🎚️ Import a library file

Point Spinlist at the folder your music lives in. It reads the tags (title & artist) from your MP3/MP4/M4A files to find matches.

Folder scanning works best in Chrome or Edge. In Safari, use "Import a library file" instead. Music videos (.m4v, .mov) are picked up too and labelled 🎬 — point this at your video folder (e.g. ~/Movies) to include them.

Forget my saved library

Export your library from your DJ app, then load the file here:

  • iTunes / Music: File › Library › Export Library… (an .xml file)
  • Rekordbox: File › Export Collection in xml format
  • Serato / other: export or save a crate as .m3u / .m3u8
